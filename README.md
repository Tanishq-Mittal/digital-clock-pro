# Digital Clock – Productivity Dashboard

A Python-based productivity dashboard built from an improved Digital Clock project.

## Features
- Alarm system
- Stopwatch
- Countdown timer
- Multiple themes
- World clocks

## Tech Stack
- Python
- Tkinter
